### Node 설치 방법
- Node 공식홈페이지 다운로드 설치
    - https://nodejs.org
- NVM활용 Node 설치
  - `NVM(Node Version Manager)`은 node버전 관리 도구를 말합니다.
  
---
### test 코드 실행 법
```sh
## npm 사용시
$ npm run test
$ npm run test:watch # watch 모드 실행법

## yarn 사용시
$ yarn test
$ yarn test:watch # watch 모드 실행법
```
